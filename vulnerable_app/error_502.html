Error 502: Servidore Web Offline

Nginx está funcionando, pero no puede comunicarse con la aplicación PHP.

Posibles Soluciones:

  • 1. Verifique que el servidor PHP esté corriendo:
    php -S 0.0.0.0:8000 -t vulnerable_app/
  • 2. Revise el archivo .env y asegúrese de que las IPs sean correctas.
  • 3. Asegúrese de que el Firewall de Ubuntu permita el tráfico en el puerto 8000.
